# Projeto-Comercial-Pedidos
Desenvolva um sistema capaz de auxiliar uma empresa na emissão de pedidos de produtos. A área de atuação da empresa, bem como os produtos por ela comercializados são de sua livre escolha.

## Desenvolva:
- Cadastros de produtos, categorias e clientes, com opções para alterações e exclusões.
- Geração de pedidos, com opção para criar novo pedido, gerenciar pedido (alteração e exclusão) e fechar pedido (atualizar estoque).
- Impressão em relatório para produtos e clientes.
- Impressão de pedidos por faixa de datas.
- Impressão de pedido individual (opcional).
